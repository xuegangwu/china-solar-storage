# ⚙️ 投资地图参数配置 - 快速入门

## 🎯 一句话说明

投资地图采用**多维度加权评分系统**，您可以自定义各维度权重，生成符合自己投资策略的评估结果。

---

## 🚀 3 分钟上手

### 第 1 步：访问配置页面

**方式 1**: 首页底部导航 → **⚙️ 参数配置**  
**方式 2**: 直接访问 http://localhost:3000/config.html

### 第 2 步：了解评分机制

投资地图评估每个省份的**4 个维度**：

| 维度 | 说明 | 默认权重 |
|------|------|----------|
| ☀️ **资源禀赋** | 光照资源、年利用小时数 | 25% |
| 💰 **电价政策** | 燃煤基准价、峰谷价差 | 25% |
| 📈 **市场条件** | 市场化比例、消纳能力 | 25% |
| ⚡ **电网条件** | 电网接入、消纳空间 | 25% |

**综合得分** = 资源 × 25% + 电价 × 25% + 市场 × 25% + 电网 × 25%

### 第 3 步：调整权重

1. **输入新权重**（0-100 之间的数字）
2. **查看权重总和**（必须 = 100%）
3. **进度条提示**：
   - 🟢 绿色 = 100%（可以保存）
   - 🟠 橙色 ≠ 100%（需要调整）
   - 🔴 红色 > 100%（超出限制）

### 第 4 步：保存配置

1. 点击 **"💾 保存配置"** 按钮
2. 等待保存成功提示
3. 自动跳转到投资地图查看新结果

## 💡 快速配置方案

### 我是资源导向型投资者（如央企）
```
资源禀赋：40%
电价政策：20%
市场条件：20%
电网条件：20%
```
**特点**: 重视长期资源价值，适合大型基地项目

### 我是政策导向型投资者（如民营企业）
```
资源禀赋：20%
电价政策：40%
市场条件：20%
电网条件：20%
```
**特点**: 关注电价政策和投资收益，回报敏感型

### 我是市场导向型投资者（如金融机构）
```
资源禀赋：20%
电价政策：20%
市场条件：40%
电网条件：20%
```
**特点**: 重视市场消纳和风险控制，稳健型

### 我是电网导向型投资者（如 EPC 企业）
```
资源禀赋：20%
电价政策：20%
市场条件：20%
电网条件：40%
```
**特点**: 关注电网接入条件，工程导向型

### 我有选择困难症
```
使用默认配置（25% + 25% + 25% + 25%）
```
**特点**: 各维度均衡，不会出错

## 📊 投资等级说明

根据综合得分，省份被划分为 4 个等级：

| 等级 | 得分 | 建议 | 颜色 |
|------|------|------|------|
| **A 级** | ≥ 80 分 | 优先投资 | 🟢 |
| **B 级** | 70-79 分 | 推荐投资 | 🔵 |
| **C 级** | 60-69 分 | 谨慎投资 | 🟡 |
| **D 级** | < 60 分 | 暂不建议 | 🔴 |

## ⚠️ 注意事项

### ✅ 正确做法
- 权重总和必须 = 100%
- 保存后刷新地图查看效果
- 多尝试不同配置对比结果

### ❌ 错误做法
- 权重总和不等于 100%（无法保存）
- 单个权重超过 100%（无效）
- 保存后不刷新地图（看不到新结果）

---

## 🔧 常见问题

### Q1: 为什么保存按钮是灰色的？
**A**: 权重总和不等于 100%，请调整到正好 100%

### Q2: 调整后地图没变化？
**A**: 保存后需要刷新地图页面，或等待自动跳转

### Q3: 可以保存多套配置吗？
**A**: 当前版本只支持一套配置，建议截图保存不同方案

### Q4: 如何恢复默认配置？
**A**: 点击"🔄 重置默认值"按钮，然后保存

### Q5: 配置会影响其他人吗？
**A**: 是的，配置是全局的，会影响所有用户看到的结果
