## 💰 ROI 引擎升级

### 当前 vs 目标

**当前**:
```javascript
{
  payback_years: 10.5,
  irr_approx: 9.5
}
```

**目标**:
```javascript
{
  payback_static: 10.5,      // 静态回收期
  payback_dynamic: 12.3,     // 动态回收期
  irr: 9.5,                  // IRR
  npv: 2500,                 // 净现值
  lcoe: 0.35,                // 度电成本
  total_revenue_25y: 8500    // 25 年累计收益
}
```

### 输入参数

```javascript
{
  capacity_kw: 100000,        // 装机容量 kW
  capacity_kwh: 200000,       // 储能容量 kWh
  unit_cost_w: 3.5,           // 光伏单位投资 元/W
  unit_cost_wh: 1.5,          // 储能单位投资 元/Wh
  loan_rate: 0.045,           // 融资利率 4.5%
  self_use_rate: 0.6,         // 自用电比例 60%
  peak_valley_spread: 0.85    // 峰谷价差 元/kWh
}
```
