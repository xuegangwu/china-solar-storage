# 🗺️ 光储电站投资地图系统

Solar-Storage Investment Map System

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## 📖 项目简介

本系统提供中国、越南、日本等国家的光储电站投资评估地图，基于多维度评分系统帮助用户快速识别优质投资区域。

---

## ✨ 核心功能

- 🗺️ **投资地图可视化**: 交互式地图展示各国省份投资评分
- 📊 **多维度评估**: 资源禀赋、电价政策、市场条件、电网条件
- 🎯 **投资等级划分**: A/B/C/D四级投资建议
- ⚙️ **参数可配置**: 支持自定义各维度权重
- 📱 **移动端适配**: 支持手机、平板、桌面端

## 🎯 评分机制

### 四大维度

```
综合得分 = 资源×25% + 电价×25% + 市场×25% + 电网×25%
```

| 维度 | 权重 | 说明 |
|------|------|------|
| ☀️ 资源禀赋 | 25% | 光照资源、年利用小时数 |
| 💰 电价政策 | 25% | 燃煤基准价、峰谷价差 |
| 📈 市场条件 | 25% | 市场化比例、消纳能力 |
| ⚡ 电网条件 | 25% | 电网接入、消纳空间 |

### 投资等级

| 等级 | 得分 | 投资建议 | 颜色 |
|------|------|----------|------|
| **A 级** | ≥ 80 分 | 优先投资 | 🟢 |
| **B 级** | 70-79 分 | 推荐投资 | 🔵 |
| **C 级** | 60-69 分 | 谨慎投资 | 🟡 |
| **D 级** | < 60 分 | 暂不建议 | 🔴 |

## 🔧 API 接口

### 获取省份数据

```bash
GET /api/provinces
```

**响应示例**:
```json
{
  "provinces": [
    {
      "id": 1,
      "name": "上海市",
      "country": "CN",
      "scores": {
        "total": 83.5,
        "resource": 70,
        "price": 90,
        "market": 85,
        "grid": 90
      },
      "grade": "B"
    }
  ]
}
```

### 获取配置

```bash
GET /api/settings
```

### 更新配置

```bash
PUT /api/settings/batch
```
