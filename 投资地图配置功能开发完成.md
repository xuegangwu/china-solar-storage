## 🎯 评分机制

### 四大评估维度

| 维度 | 默认权重 | 说明 | 评估指标 |
|------|----------|------|----------|
| **☀️ 资源禀赋** | 25% | 光照资源、年利用小时数 | 辐射量、小时数 |
| **💰 电价政策** | 25% | 燃煤基准价、峰谷价差 | 电价水平、价差 |
| **📈 市场条件** | 25% | 市场化比例、消纳能力 | 交易比例、消纳 |
| **⚡ 电网条件** | 25% | 电网接入、消纳空间 | 接入能力、弃光率 |

### 评分公式

```
Total Score = Resource × W₁ + Price × W₂ + Market × W₃ + Grid × W₄

约束条件：W₁ + W₂ + W₃ + W₄ = 100%
```

### 投资等级

| 等级 | 得分范围 | 投资建议 | 颜色 |
|------|----------|----------|------|
| **A 级** | ≥ 80 分 | 优先投资 | 🟢 |
| **B 级** | 70-79 分 | 推荐投资 | 🔵 |
| **C 级** | 60-69 分 | 谨慎投资 | 🟡 |
| **D 级** | < 60 分 | 暂不建议 | 🔴 |

## 🚀 使用流程

### 1. 访问配置页面

**方式一**: 首页底部导航
```
首页 → ⚙️ 参数配置
```

**方式二**: 直接访问
```
http://localhost:3000/config.html
```

### 2. 调整权重

1. 修改各维度权重值（0-100）
2. 实时查看权重总和（必须 = 100%）
3. 进度条显示当前总和状态

**视觉效果**:
- ✅ 总和 100%：绿色进度条，可保存
- ⚠️ 总和 < 100%：橙色进度条，不可保存
- ❌ 总和 > 100%：红色进度条，不可保存

### 3. 保存配置

1. 点击"💾 保存配置"按钮
2. 系统验证权重总和
3. 保存到数据库
4. 自动跳转到投资地图

### 4. 查看效果

保存后自动跳转到投资地图页面，查看新的评分结果。

### 后端 API

**接口**: `PUT /api/settings/batch`

**请求**:
```http
PUT /api/settings/batch
Authorization: Bearer <token>
Content-Type: application/json

{
  "weight_resource": "25",
  "weight_price": "25",
  "weight_market": "25",
  "weight_grid": "25"
}
```

**响应**:
```json
{
  "message": "设置批量更新成功"
}
```

**表名**: `system_settings`

**字段**:
- `key_name`: 配置键名
- `value`: 配置值
- `description`: 说明
- `updated_at`: 更新时间

## ⚠️ 注意事项

### 1. 权限要求

- 配置页面需要登录
- 保存操作需要 admin 权限
- Token 存储在 localStorage

### 2. 权重验证

- 总和必须 = 100%
- 单个权重范围：0-100
- 不满足条件时保存按钮禁用

### 3. 数据更新

- 保存后地图自动更新
- 需要刷新地图页面查看新结果
- 建议保存前导出当前数据
