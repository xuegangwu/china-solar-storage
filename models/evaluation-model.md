# 投资评估模型 v0.2

## 核心变化

**消纳条件权重提升至 50%** - 这是当前光储投资的核心瓶颈

## 评估公式

```
总分 = 资源条件×0.15 + 电价水平×0.20 + 政策支持×0.15 + 消纳条件×0.50
```

## 权重配置（可调）

| 维度 | 默认权重 | 可调范围 | 说明 |
|-----|---------|---------|------|
| 资源条件 | 15% | 0-50% | 光照资源差异相对较小 |
| 电价水平 | 20% | 0-50% | 收益核心，含峰谷价差 |
| 政策支持 | 15% | 0-50% | 补贴退坡，影响降低 |
| **消纳条件** | **50%** | 20-80% | **关键瓶颈，一票否决** |

## 评分标准

### 1. 资源条件 (0-100 分)

| 年等效利用小时 | 得分 |
|---------------|------|
| ≥1500h | 100 |
| 1300-1500h | 80 |
| 1100-1300h | 60 |
| 900-1100h | 40 |
| <900h | 20 |

### 2. 电价水平 (0-100 分)

**实际电价 = 燃煤基准价 × 市场化系数**

| 实际电价 (元/kWh) | 得分 |
|-------------------|------|
| ≥0.45 | 100 |
| 0.40-0.45 | 80 |
| 0.35-0.40 | 60 |
| 0.30-0.35 | 40 |
| <0.30 | 20 |

**峰谷价差加分** (最高 +25 分):
- 峰谷价差≥0.8 元：+25 分
- 峰谷价差 0.6-0.8 元：+20 分
- 峰谷价差 0.4-0.6 元：+15 分
- 峰谷价差 0.3-0.4 元：+10 分
- 峰谷价差<0.3 元：+5 分

### 3. 政策支持 (0-100 分)

基础分 50 分，根据政策叠加（上限 100 分）:

| 政策类型 | 分值 |
|---------|------|
| 省级补贴 | +30 |
| 市级补贴 | +15 |
| 税收优惠 | +15 |
| 土地优惠 | +10 |
| 消纳保障 | +20 |
| 审批绿色通道 | +10 |

### 4. 消纳条件 (0-100 分) ⭐核心指标

| 弃光率 | 得分 |
|-------|------|
| <2% | 100 |
| 2-4% | 80 |
| 4-6% | 60 |
| 6-10% | 40 |
| 10-15% | 20 |
| >15% | 10 |

**电网容量系数调节**:
- 最终得分 = 弃光率得分 × 电网剩余容量比例
- 电网容量紧张地区进一步扣分

## 投资等级判定

| 总分 | 等级 | 建议 |
|-----|------|------|
| 85-100 | A | 优先投资 |
| 70-84 | B | 推荐投资 |
| 55-69 | C | 谨慎投资 |
| <55 | D | 暂不建议 |

## ROI 估算模型

```
实际发电小时 = 年等效利用小时 × (1 - 弃光率)
年收入 = 装机容量 × 1000 × 实际发电小时 × 实际电价 / 10000
年运维成本 = 总投资 × 2%
年净收益 = 年收入 - 年运维成本
投资回收期 = 总投资 / 年净收益
估算 IRR = 年净收益 / 总投资 × 100%
```

**默认参数**:
- 单位投资：350 万元/MW
- 运营期：25 年
- 运维成本：投资额的 2%/年
