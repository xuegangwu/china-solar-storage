# ✅ 中国投资地图过滤越南省份

**修改时间**: 2026-03-12 13:40  
**修改人**: Javis

---

## 🎯 修改目标

用户要求：中国投资地图只显示中国省份，不包含越南的投资省市信息。

---

## 🔍 问题分析

### 当前数据状况

后端数据库包含 45 个省份：
- **中国省份**: 30 个
- **越南省份**: 15 个

### 问题表现

中国投资地图（`map-v1.0.html`）显示所有 45 个省份，包括：
- 中国省份（上海、浙江、广东等）
- 越南省份（宁顺省、平顺省、嘉莱省等）

**影响**:
- 用户混淆：中国地图中出现越南地名
- 数据不准确：统计数据包含越南
- 体验不佳：需要手动区分国家

---

## ✅ 修改方案

### 前端过滤（推荐）

**文件**: `web/map-v1.0.html`

**修改位置**: `loadProvinces()` 函数

**修改前**:
```javascript
if (data.provinces && data.provinces.length > 0) {
    provincesData = data.provinces;
    filteredProvinces = [...provincesData];
    renderProvinceList();
    renderMap();
    updateStats();
}
```

**修改后**:
```javascript
if (data.provinces && data.provinces.length > 0) {
    provincesData = data.provinces.filter(p => p.country === 'CN'); // ✅ 只显示中国省份
    filteredProvinces = [...provincesData];
    renderProvinceList();
    renderMap();
    updateStats();
}
```

**说明**:
- 使用 `filter()` 方法筛选 `country === 'CN'` 的省份
- 只保留中国省份数据
- 自动更新统计和地图显示
