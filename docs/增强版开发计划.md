## 🏗️ 技术架构

### 后端技术栈
```
Node.js + Express
├── PostgreSQL (主数据库)
├── Redis (缓存)
├── JWT (身份验证)
├── Swagger (API 文档)
└── Nodemailer (邮件服务)
```

### 前端增强
```
React (可选升级)
├── Axios (HTTP 客户端)
├── React Query (数据同步)
├── Zustand (状态管理)
└── React Hook Form (表单处理)
```

### AI 服务
```
Python + FastAPI
├── Scikit-learn (机器学习)
├── Pandas (数据处理)
├── NumPy (数值计算)
└── Joblib (模型保存)
```
