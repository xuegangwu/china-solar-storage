## 🚀 使用方式

### 登录系统

```bash
# 打开登录页面
open web/login.html
```

**操作流程**:
1. 打开登录页面
2. 输入用户名和密码
3. 点击登录
4. 自动跳转（管理员→后台，用户→地图）

### 后台管理

```bash
# 打开后台管理（需要管理员权限）
open web/admin.html
```

**前提条件**:
- 已登录且角色为 admin
- 未登录会自动跳转到登录页

### 场景 1：管理员日常

1. 后台 → 系统设置
2. 调整评估权重

4. 保存设置

### 场景 4：数据备份
1. 后台 → 数据管理
2. 点击"导出全部数据"
3. 下载 JSON 文件
4. 保存备份
