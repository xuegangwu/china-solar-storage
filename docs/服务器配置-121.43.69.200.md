### 4. 配置环境变量

```bash

```

**需要配置的内容：**
```bash

PORT=3001

# OSS_ACCESS_KEY_SECRET=<你的密钥>
```

### 5. 初始化数据库

```bash

```

### 6. 配置域名（可选）

```bash
# 编辑 Nginx 配置
nano /etc/nginx/sites-available/solar-storage-map

# 修改 server_name 为你的域名
```

### 7. 重启服务


## 🔒 安全配置

### 配置防火墙

```bash
# 安装 UFW
apt install -y ufw

# 配置规则
ufw allow 22/tcp    # SSH
ufw allow 80/tcp    # HTTP
ufw allow 443/tcp   # HTTPS
ufw allow 3001/tcp  # API

# 启用防火墙
ufw enable
ufw status
```

### 配置 SSH 安全

```bash
# 编辑 SSH 配置
nano /etc/ssh/sshd_config

# 建议修改：
# Port 2222              # 修改 SSH 端口
# PermitRootLogin no     # 禁止 root 登录
# PasswordAuthentication no  # 禁用密码登录

# 重启 SSH 服务
systemctl restart sshd
```
