## 🌏 区域市场

### 1. map-v1.0.html - 中国区地图
**路径：** `/map-v1.0.html`  
**功能：** 中国 30 省份地图展示、评分、排名

**页面结构：**
```
- 交互式 SVG 地图
- 省份评分热力图
- 排名列表
- 筛选器（等级、分数）
- 搜索框
- 省份卡片（点击跳转详情）
```

**关联页面：**
- → `province-detail.html?id=X` 省份详情
- → `calculator.html` 收益计算
- → `comparison.html` 对比分析

---

### 2. province-detail.html - 省份详情
**路径：** `/province-detail.html?id=1`  
**功能：** 省份详细信息、多维度分析

**页面结构：**
```
- 返回导航
- 省份基本信息（名称、等级、得分）
- 概览统计（4 个指标卡片）
- 多维度雷达图
- 历史趋势图
- 详细参数列表（10+ 指标）
- 投资建议
- 操作按钮（收藏、导出、计算收益、对比）
```

**URL 参数：**
- `id` - 省份 ID（1-30）

---

### 3. europe-map.html - 欧洲区地图
**路径：** `/europe-map.html`  
**功能：** 欧洲 8 国列表、区域统计

**页面结构：**
```
- 区域选择器（中国/欧洲/东南亚）
- 区域统计（4 个指标）
- 国家列表（8 张卡片）
  - 国旗、名称、得分、等级
  - 关键指标（资源、电价、政策、利用小时、补贴）
  - 点击查看详情
```

**关联页面：**
- → `europe-country-detail.html?id=101` 国家详情
- → `map-v1.0.html` 切换中国区

---

### 4. europe-country-detail.html - 欧洲国家详情
**路径：** `/europe-country-detail.html?id=101`  
**功能：** 欧洲国家详细信息

**页面结构：**
```
- 返回欧洲地图
- 国家基本信息（国旗、名称、首都、区域）
- 概览统计（得分、等级、欧盟成员、区域）

- 操作按钮（收藏、对比、计算收益）
```

**URL 参数：**
- `id` - 国家 ID（101-108）

## 🛠️ 分析工具

### 1. calculator.html - 收益计算器
**路径：** `/calculator.html`  
**功能：** 投资收益计算、20 年现金流预测

**页面结构：**
```
- 投资参数输入
  - 省份选择
  - 投资规模（万元）
  - 装机容量（kW）
  - 储能容量（kWh）
  - 年利用小时
  - 电价
  - 运营成本占比
- 计算结果
  - 年发电量
  - 年收入/成本/收益
  - 投资回收期
  - 内部收益率 (IRR)
  - 10 年现金流图表
```

---

### 2. comparison.html - 对比分析
**路径：** `/comparison.html`  
**功能：** 多省份/国家对比

**页面结构：**
```
- 已选省份列表（最多 3 个）
- 对比图表
  - 综合评分柱状图
  - 多维度雷达图
  - 历史趋势对比
  - 详细数据对比
- 移除省份按钮
- 添加省份按钮
```

**数据来源：**
- LocalStorage 存储对比列表

---

### 3. report-generator.html - 报告生成
**路径：** `/report-generator.html`  
**功能：** 生成专业投资分析报告

**页面结构：**
```
- 步骤 1：选择省份（多选）
- 步骤 2：选择报告模板
  - 标准分析报告（5 页）
  - 详细分析报告（15 页）
  - 决策简报（3 页）
  - 多省对比报告（8 页）
- 步骤 3：配置选项
  - 包含图表
  - 包含投资建议
  - 包含历史数据
  - 邮件发送副本
- 生成按钮
- 下载链接
```

**API 调用：**
- `POST /api/reports/generate`
- `GET /api/reports/:id/download`

## 🎯 用户旅程

### 旅程 1：投资分析
```
1. 访问门户首页
2. 选择区域（中国/欧洲）
3. 查看地图和排名
4. 点击感兴趣的省份/国家
5. 查看详细分析
6. 使用收益计算器
7. 添加到对比列表
8. 生成分析报告
```

### 旅程 2：多省对比
```
1. 访问门户首页
2. 进入中国区地图
3. 查看多个省份
4. 点击"对比"按钮
5. 进入对比分析页面
6. 查看对比图表
7. 调整对比省份
8. 保存对比结果
```

### 旅程 3：接收通知
```
1. 登录系统
2. 订阅通知类型
3. 后台监控政策变化
4. 收到通知推送
5. 访问通知中心
6. 查看详情
7. 标记已读
```
