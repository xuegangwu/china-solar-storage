# 数据采集计划

## 第一阶段：手动收集 (v0.1)

### 目标省份
优先收集以下 10 个重点省份：
1. 山东 - 工商业储能大省
2. 江苏 - 经济发达，电价高
3. 浙江 - 峰谷价差大
4. 广东 - 用电负荷大
5. 河北 - 光伏装机大省
6. 山西 - 能源转型
7. 安徽 - 中部代表
8. 湖北 - 中部代表
9. 四川 - 西南代表
10. 甘肃 - 西北代表

### 采集内容

#### 1. 电站项目
- 项目名称
- 位置（省市区）
- 装机规模
- 状态（存量/在建/规划）
- 开发商
- 并网时间（如已知）

#### 2. 政策数据
- 燃煤基准价
- 峰谷电价政策
- 补贴政策
- 其他优惠政策

## 第二阶段：自动化采集 (v0.2)

### 数据源

1. **政府网站**
   - 国家能源局：http://www.nea.gov.cn
   - 各省发改委/能源局
   
2. **行业媒体**
   - 光伏們：https://www.pvmen.com
   - 储能头条：https://www.esmedia.com.cn
   - 北极星电力网：https://www.bjx.com.cn

3. **招投标平台**
   - 中国招标投标公共服务平台
   - 各省公共资源交易中心

### 技术实现
- Python Scrapy 爬虫
- 定期运行（每周）
- 数据去重和验证

## 第三阶段：数据合作 (v0.3+)

探索与以下机构合作：
- 行业协会
- 咨询公司
- 数据服务商

## 数据验证

- 交叉验证多个来源
- 优先采用官方数据
- 标注数据置信度

## 注意事项

1. 遵守 robots.txt
2. 控制爬取频率
3. 数据仅供研究使用
4. 标注数据来源
