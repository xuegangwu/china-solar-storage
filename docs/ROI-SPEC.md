# 光储电站投资 ROI 计算规格说明书

> 版本：v1.0  
> 日期：2026-03-11  
> 用途：光储电站投资决策系统

## 1. 概述

### 1.1 计算目标
为光储电站投资提供量化的财务评估指标，包括：
- 静态投资回收期
- 动态投资回收期
- 内部收益率（IRR）
- 净现值（NPV）
- 度电成本（LCOE）
- 25 年累计收益

### 1.2 适用范围
- 集中式光伏电站
- 工商业分布式光伏
- 光储一体化项目
- 独立储能电站

## 2. 输入参数

### 2.1 基础参数

| 参数 | 符号 | 单位 | 默认值 | 说明 |
|-----|------|------|--------|------|
| 光伏装机容量 | P_pv | kW | 100,000 | 100MW |
| 储能装机容量 | E_bess | kWh | 200,000 | 200MWh |
| 运营年限 | N | 年 | 25 | 项目寿命 |
| 年等效利用小时 | H | 小时 | 1,200 | 当地光照资源 |
| 系统效率 | η | % | 85% | 综合效率 |

### 2.2 投资成本

| 参数 | 符号 | 单位 | 默认值 | 说明 |
|-----|------|------|--------|------|
| 光伏单位投资 | C_pv | 元/W | 3.5 | 含组件、逆变器等 |
| 储能单位投资 | C_bess | 元/Wh | 1.5 | 含电池、PCS 等 |
| 土地成本 | C_land | 万元 | 500 | 一次性 |
| 其他费用 | C_other | 万元 | 1,000 | 设计、监理等 |

**总投资计算**:
```
C_total = P_pv × 1000 × C_pv + E_bess × C_bess + C_land × 10000 + C_other × 10000
```

### 2.3 电价参数

| 参数 | 符号 | 单位 | 说明 |
|-----|------|------|------|
| 燃煤基准价 | P_base | 元/kWh | 各省不同 |
| 市场化交易系数 | k_market | - | 0.85-0.95 |
| 峰段电价 | P_peak | 元/kWh | 峰段电价 |
| 谷段电价 | P_valley | 元/kWh | 谷段电价 |
| 峰谷价差 | ΔP | 元/kWh | P_peak - P_valley |
| 自用电比例 | r_self | % | 自发自用比例 |

**实际电价计算**:
```
P_actual = P_base × k_market
```

### 2.4 政策参数

| 参数 | 符号 | 单位 | 说明 |
|-----|------|------|------|
| 省级补贴 | S_province | 元/kWh | 省级度电补贴 |
| 市级补贴 | S_city | 元/kWh | 市级度电补贴 |
| 补贴年限 | N_sub | 年 | 补贴期限 |
| 企业所得税率 | r_tax | % | 一般 25% |
| 增值税率 | r_vat | % | 一般 13% |

### 2.5 消纳参数

| 参数 | 符号 | 单位 | 默认值 | 说明 |
|-----|------|------|--------|------|
| 弃光率 | r_curtailed | % | 3-15% | 弃光比例 |
| 电网容量系数 | k_grid | - | 0.6-0.9 | 电网接纳能力 |

**有效发电小时**:
```
H_effective = H × (1 - r_curtailed)
```

### 2.6 融资参数

| 参数 | 符号 | 单位 | 默认值 | 说明 |
|-----|------|------|--------|------|
| 资本金比例 | r_equity | % | 30% | 自有资金 |
| 贷款比例 | r_debt | % | 70% | 银行贷款 |
| 贷款利率 | i_loan | % | 4.5% | 年利率 |
| 贷款期限 | N_loan | 年 | 15 | 还款期限 |

## 3. 收入计算

### 3.1 年发电量

**首年发电量**:
```
E_year1 = P_pv × H_effective × η / 1000  (万 kWh)
```

**衰减后发电量** (第 t 年):
```
E_t = E_year1 × (1 - d)^(t-1)
```

其中：
- d = 组件衰减率，默认 0.5%/年

### 3.2 售电收入

**基础售电收入**:
```
R_base_t = E_t × P_actual × 10000  (万元)
```

**补贴收入** (补贴期内):
```
R_sub_t = E_t × (S_province + S_city) × 10000  (万元)  [t ≤ N_sub]
R_sub_t = 0  [t > N_sub]
```

**储能套利收入**:
```
R_arbitrage = E_bess × ΔP × 365 × η_round / 10000  (万元/年)
```

其中：
- η_round = 储能往返效率，默认 85%

**总年收入**:
```
R_t = R_base_t + R_sub_t + R_arbitrage
```

### 3.3 成本计算

**运维成本**:
```
C_om_t = C_total × r_om  (万元/年)
```

其中：
- r_om = 运维费率，默认 2%/年

**贷款还本付息** (等额本息):
```
A_loan = L × i_loan × (1 + i_loan)^N_loan / [(1 + i_loan)^N_loan - 1]
```

其中：
- L = 贷款总额 = C_total × r_debt

**总年成本**:
```
C_t = C_om_t + A_loan  [t ≤ N_loan]
C_t = C_om_t  [t > N_loan]
```

### 3.4 税金计算

**增值税**:
```
T_vat_t = R_t × r_vat
```

**企业所得税**:
```
T_income_t = (R_t - C_t - T_vat_t) × r_tax × (1 - r_tax_reduction)
```

其中：
- r_tax_reduction = 税收优惠比例，三免三减半为 50%

**净利润**:
```
Profit_t = R_t - C_t - T_vat_t - T_income_t
```

## 4. 财务指标计算

### 4.1 静态投资回收期

**定义**: 累计净收益等于总投资的年限

**计算**:
```
Payback_static = min{t | Σ(Profit_i) ≥ C_total}
```

**简化公式** (年均净收益):
```
Payback_static ≈ C_total / Annual_Average_Profit
```

**合理范围**: 8-12 年

### 4.2 动态投资回收期

**定义**: 考虑资金时间价值的回收期

**计算**:
```
Payback_dynamic = min{t | Σ[Profit_i / (1 + r)^i] ≥ C_total}
```

其中：
- r = 折现率，默认 8%

**合理范围**: 10-15 年

### 4.3 内部收益率 (IRR)

**定义**: 使净现值为零的折现率

**计算**:
```
NPV = -C_total + Σ[Profit_t / (1 + IRR)^t] = 0
```

**迭代求解**:
```python
def calculate_irr(cash_flows):
    from scipy.optimize import brentq
    def npv(r):
        return sum([cf / (1 + r)**t for t, cf in enumerate(cash_flows)])
    return brentq(npv, -0.99, 1.0)
```

**合理范围**: 8-12%

### 4.4 净现值 (NPV)

**定义**: 未来现金流折现值减去初始投资

**计算**:
```
NPV = -C_total + Σ[Profit_t / (1 + r)^t]
```

其中：
- r = 基准收益率，默认 8%

**判断标准**:
- NPV > 0: 项目可行
- NPV < 0: 项目不可行

### 4.5 度电成本 (LCOE)

**定义**: 项目生命周期内单位发电量的成本

**计算**:
```
LCOE = [C_total + Σ(C_t / (1 + r)^t)] / Σ[E_t / (1 + r)^t]
```

单位：元/kWh

**合理范围**: 0.25-0.40 元/kWh

### 4.6 25 年累计收益

**计算**:
```
Total_Revenue_25y = Σ(Profit_t)  [t=1 to 25]
```

单位：万元

## 5. 输出指标

### 5.1 核心指标

| 指标 | 符号 | 单位 | 合理范围 | 权重 |
|-----|------|------|---------|------|
| 静态回收期 | Payback_static | 年 | 8-12 | 25% |
| 动态回收期 | Payback_dynamic | 年 | 10-15 | 15% |
| 内部收益率 | IRR | % | 8-12% | 30% |
| 净现值 | NPV | 万元 | >0 | 15% |
| 度电成本 | LCOE | 元/kWh | 0.25-0.40 | 15% |

### 5.2 辅助指标

| 指标 | 说明 |
|-----|------|
| 总投资 | C_total (万元) |
| 年发电量 | E_year1 (万 kWh) |
| 年收入 | R_year1 (万元) |
| 年净收益 | Profit_year1 (万元) |
| 25 年累计收益 | Total_Revenue_25y (万元) |
| 资本金收益率 | ROE (%) |

### 5.3 评分计算

**ROI 评分** (0-100 分):
```
Score_ROI = 25 × S_payback + 15 × S_dynamic + 30 × S_irr + 15 × S_npv + 15 × S_lcoe
```

**分项评分**:

| 指标 | 评分标准 |
|-----|---------|
| 回收期 | <8 年:100 分, 8-10 年:80 分, 10-12 年:60 分, >12 年:40 分 |
| IRR | >12%:100 分, 10-12%:80 分, 8-10%:60 分, <8%:40 分 |
| NPV | >5000 万:100 分, 2000-5000 万:80 分, 0-2000 万:60 分, <0:40 分 |

## 6. 计算示例

### 6.1 案例参数

**项目**: 浙江省 100MW 光储项目

| 参数 | 值 |
|-----|-----|
| 光伏装机 | 100,000 kW |
| 储能装机 | 200,000 kWh |
| 光伏单位投资 | 3.5 元/W |
| 储能单位投资 | 1.5 元/Wh |
| 年等效小时 | 1,100 h |
| 燃煤电价 | 0.4155 元/kWh |
| 弃光率 | 2% |
| 省级补贴 | 0.1 元/kWh (3 年) |

### 6.2 计算结果

**投资**:
```
C_total = 100,000 × 1000 × 3.5 + 200,000 × 1.5 = 38,000 万元
```

**年发电量**:
```
E_year1 = 100,000 × 1100 × 0.98 × 0.85 / 1000 = 9,163 万 kWh
```

**年收入**:
```
R_base = 9,163 × 0.4155 = 3,807 万元
R_sub = 9,163 × 0.1 = 916 万元 (前 3 年)
R_arbitrage = 200,000 × 0.4155 × 365 × 0.85 / 10000 = 2,579 万元
R_total = 3,807 + 916 + 2,579 = 7,302 万元 (前 3 年)
```

**财务指标**:
| 指标 | 结果 | 评级 |
|-----|------|------|
| 静态回收期 | 9.2 年 | B |
| 动态回收期 | 11.5 年 | B |
| IRR | 10.8% | B |
| NPV (8%) | 2,850 万元 | B |
| LCOE | 0.32 元/kWh | A |
| 25 年累计收益 | 42,500 万元 | - |

**综合评分**: 78 分 (B 级 - 推荐投资)

## 8. 假设与限制

### 8.1 基本假设

1. 项目运营期 25 年
2. 组件衰减率 0.5%/年
3. 运维成本 2%/年递增
4. 电价保持不变（不考虑上涨）
5. 无重大政策变化

### 8.2 限制条件

1. 不考虑土地增值税
2. 不考虑碳交易收益
3. 不考虑绿证收益
4. 不考虑融资约束
5. 不考虑极端天气影响

### 8.3 风险提示

⚠️ **重要提示**:
- 本计算结果仅供参考
- 实际投资需结合实地调研
- 电价政策可能变化
- 消纳条件可能恶化
- 建议进行敏感性分析
