# v0.4 发布说明

> 发布时间：2026-03-11  
> 版本：v0.4 - 交互式地图版

## 🎉 新功能

### 1. Leaflet 交互式地图

**核心功能**:
- ✅ 中国地图可视化展示
- ✅ 省份标记（按投资等级着色）
- ✅ 点击弹窗显示详情
- ✅ 点击省份自动定位
- ✅ 地图图例

**标记样式**:
- 🔵 B 级（推荐投资）- 蓝色
- 🟡 C 级（谨慎投资）- 黄色
- 🔴 D 级（暂不建议）- 红色
- 标记大小与总分成正比

**底图**: CartoDB Positron（简洁风格）

### 2. 数据导出功能

**支持格式**:
- 📊 JSON - 完整数据结构
- 📄 CSV - Excel 可打开

**导出内容**:
- 省份基本信息
- 四项评分（资源/电价/政策/消纳）
- ROI 数据（投资/收入/回收期/IRR）

**使用方式**:
```javascript
// 页面右上角按钮
导出 JSON → china-solar-storage-provinces.json
导出 CSV → china-solar-storage-provinces.csv
```

### 3. 增强侧边栏

**新功能**:
- 🔍 省份搜索框
- 📊 实时统计卡片
- 📋 省份列表（可排序）
- 📖 详情面板（选中省份）

**筛选功能**:
- 按投资等级筛选（全部/B/C/D）
- 搜索框实时过滤
- 列表与地图联动

### 4. 响应式布局

**改进**:
- 侧边栏可隐藏/显示
- 地图自适应窗口大小
- 移动端友好（基础支持）

## 🚀 使用方式

### 查看地图

```bash
# 浏览器打开
open web/map.html

# 或访问
file:///workspace/projects/china-solar-storage/web/map.html
```

### 导出数据

1. 打开 `web/map.html`
2. 点击右上角"导出 JSON"或"导出 CSV"
3. 文件自动下载

### 功能操作

| 操作 | 方式 |
|-----|------|
| 查看省份详情 | 点击地图标记或列表项 |
| 筛选等级 | 点击筛选按钮（全部/B/C/D） |
| 搜索省份 | 侧边栏搜索框输入 |
| 隐藏侧边栏 | 点击"列表"按钮 |
| 导出数据 | 点击右上角导出按钮 |

2. **移动端**: 侧边栏在手机上占用空间较大
3. **地图缩放**: 最远缩放级别有限
