## 一、电价数据

### 1.1 燃煤基准价（固定）

**来源**: 国家发改委《关于深化燃煤发电上网电价形成机制改革的指导意见》

**获取方式**: 
- 手动整理（一次性）
- 文件：`policy/benchmark_prices.csv`

**各省基准价参考**（2020 年）:
```
山东：0.3949 元/kWh
江苏：0.3910 元/kWh
浙江：0.4155 元/kWh
广东：0.4530 元/kWh
...
```

### 1.2 峰谷电价（动态）

**来源**: 各省发改委《关于完善分时电价机制的通知》

**获取方式**:
- 爬虫：各省发改委官网
- 脚本：`scripts/scrape_price_policy.py`

**关键信息**:
- 峰段、平段、谷段时间划分
- 各时段电价
- 峰谷价差

### 1.3 市场化交易电价

**来源**: 各省电力交易中心月度报告

**获取方式**:
- 爬虫：电力交易中心网站
- 估算：燃煤基准价 × 0.85~0.95（根据供需）

## 二、消纳数据

### 2.1 弃光率（核心指标）

**来源**: 全国新能源电力消纳监测预警中心

**网址**: http://www.chinapower.org.cn

**获取方式**:
- 手动：季度/年度报告
- 爬虫：定期抓取

**历史数据参考**:
```
2023 年全国平均弃光率：2.0%
西北地区：5-15%
华北地区：3-8%
华东地区：1-3%
```

### 2.2 电网容量系数（估算）

**公式**: `电网容量系数 = 1 - (光伏装机 / 电网最大负荷)`

**数据来源**:
- 各省光伏装机：国家能源局月度统计
- 电网最大负荷：各省电网公司报告

**简化估算**:
- 装机/负荷 < 20%: 系数 0.9
- 装机/负荷 20-40%: 系数 0.7
- 装机/负荷 40-60%: 系数 0.5
- 装机/负荷 > 60%: 系数 0.3

## 三、光照资源数据

### 3.1 NASA SSE API（免费）

**网址**: https://power.larc.nasa.gov

**API 示例**:
```
https://power.larc.nasa.gov/api/temporal/daily/point?
parameters=ALLSKY_SFC_SW_DWN&
community=RE&
longitude=116.4&latitude=39.9&
start=20200101&end=20201231&
format=JSON
```

**获取**:
- 日射量 (kWh/m²/day)
- 可换算成年等效利用小时

**脚本**: `scripts/fetch_solar_resource.py`

### 3.2 中国气象局数据

**来源**: 中国气象数据网 (http://data.cma.cn)

**需要注册**, 数据更精确

## 四、电站项目数据

### 4.1 行业媒体爬取

**目标网站**:
- 光伏們 (https://www.pvmen.com)
- 储能头条 (https://www.esmedia.com.cn)
- 北极星电力网 (https://www.bjx.com.cn)

**爬取内容**:
- 项目名称
- 地点
- 规模
- 开发商
- 状态（开工/并网）

**脚本**: `scripts/scrape_projects.py`

**注意**:
- 遵守 robots.txt
- 控制爬取频率（1 请求/秒）
- 仅用于研究

### 4.2 招投标平台

**来源**:
- 中国招标投标公共服务平台
- 各省公共资源交易中心

**信息**:
- 项目备案信息
- EPC 招标
- 设备采购

## 五、政策数据

### 5.1 补贴政策

**来源**:
- 国家能源局
- 各省能源局官网

**爬取关键词**:
- "光伏补贴"
- "储能补贴"
- "分布式光伏"
- "光储一体化"

**脚本**: `scripts/scrape_policies.py`

### 5.2 其他政策

- 税收优惠：税务局文件
- 土地政策：自然资源部
- 消纳保障：能源局文件

## 六、数据验证与置信度

### 6.1 交叉验证

- 同一数据多个来源对比
- 官方数据优先
- 时间戳标注

### 6.2 置信度分级

| 等级 | 说明 | 来源 |
|-----|------|------|
| A | 官方发布 | 国家能源局、发改委 |
| B | 权威机构 | 电网公司、交易中心 |
| C | 行业媒体 | 光伏們、北极星 |
| D | 估算/推测 | 模型推算 |

### 6.3 数据标注

每个数据点标注:
```json
{
  "value": 0.3949,
  "source": "山东省发改委",
  "url": "https://...",
  "date": "2024-01-15",
  "confidence": "A"
}
```
