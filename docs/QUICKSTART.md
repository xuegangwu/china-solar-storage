# 快速使用指南

## 🗺️ 查看交互式地图

### 第一步：打开地图
```bash
# Mac
open web/map.html

# Windows
start web/map.html

# Linux
xdg-open web/map.html
```

或直接双击 `web/map.html` 文件

### 第二步：浏览地图

**地图操作**:
- 🖱️ **拖动**: 鼠标左键拖动地图
- 🔍 **缩放**: 鼠标滚轮或双击
- 📍 **查看省份**: 点击圆形标记

**标记说明**:
- 🔵 蓝色 = B 级（推荐投资）
- 🟡 黄色 = C 级（谨慎投资）
- 🔴 红色 = D 级（暂不建议）
- 标记越大 = 总分越高

### 第三步：查看详情

**方式 1 - 点击地图标记**:
1. 点击省份标记
2. 查看弹窗摘要
3. 侧边栏显示完整详情

**方式 2 - 点击列表**:
1. 左侧列表点击省份
2. 地图自动定位到该省
3. 显示详情面板

### 第四步：筛选与搜索

**按等级筛选**:
1. 点击"B 级"、"C 级"或"D 级"按钮
2. 地图和列表只显示对应等级省份
3. 点击"全部"恢复显示

**搜索省份**:
1. 在搜索框输入省份名
2. 实时过滤结果
3. 清空搜索框恢复全部

### 第五步：导出数据

**导出 JSON**:
1. 点击右上角"📊 导出 JSON"
2. 自动下载 `china-solar-storage-provinces.json`
3. 可用文本编辑器或程序打开

**导出 CSV**:
1. 点击右上角"📄 导出 CSV"
2. 自动下载 `china-solar-storage-provinces.csv`
3. 可用 Excel 打开

## 📊 使用静态页面（权重调节）

### 打开页面
```bash
open web/index.html
```

### 调节权重

1. 拖动滑块调整各维度权重
2. 权重总和必须为 100%
3. 点击"应用权重"重新计算

### ROI 测算

1. 输入装机容量（MW）
2. 输入单位投资（万元/MW）
3. 点击"重新计算"
4. 查看估算结果

## 💡 使用场景

### 场景 1：快速了解某省

1. 搜索省份名
2. 点击查看详情
3. 查看总分、分项得分、ROI

### 场景 2：筛选投资目标

1. 点击"B 级"筛选
2. 查看推荐省份列表
3. 对比各省份 ROI
4. 导出 CSV 做进一步分析

### 场景 3：团队讨论

1. 打开地图页面
2. 投影到大屏幕
3. 点击各省份查看对比
4. 导出数据分享给团队

### 场景 4：客户演示

1. 打开地图页面
2. 展示全国投资分布
3. 点击目标省份详细说明
4. 导出 PDF 报告（v0.5+）
