### 3.2 采集流程

```
1. 定时触发 (每日 8:00, 14:00, 20:00)
   ↓
2. 访问目标网站
   ↓
3. 提取政策信息 (标题、发布日期、内容、附件)
   ↓
4. NLP 处理 (分类、标签、摘要)
   ↓
5. 翻译 (非中文政策)
   ↓
6. 去重检测
   ↓
7. 存储到数据库
   ↓
8. 推送通知 (重要政策)
```

### 3.3 爬虫示例代码

```python
# 国家能源局爬虫示例
import scrapy
from datetime import datetime

class NEAPolicySpider(scrapy.Spider):
    name = 'nea_policy'
    start_urls = [
        'http://www.nea.gov.cn/zwgk/zcfg/index.htm',
        'http://www.nea.gov.cn/zwgk/zcfg/ny/index.htm'
    ]
    
    def parse(self, response):
        for item in response.css('ul.list_list li'):
            title = item.css('a::text').get()
            link = item.css('a::attr(href)').get()
            date = item.css('span::text').get()
            
            yield {
                'title': title,
                'link': response.urljoin(link),
                'publish_date': date,
                'source': '国家能源局',
                'country': 'CN',
                'category': self.categorize(title),
                'collected_at': datetime.utcnow().isoformat()
            }
    
    def categorize(self, title):
        """政策分类"""
        keywords = {
            '补贴': 'subsidy',
            '电价': 'feed_in_tariff',
            '配储': 'storage_requirement',
            '并网': 'grid_connection',
            '规划': 'planning'
        }
        for kw, cat in keywords.items():
            if kw in title:
                return cat
        return 'other'
```
