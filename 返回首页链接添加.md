## 🔧 技术细节

### 链接 vs 按钮

**使用 `<a>` 标签而非 `<button>`**:
```html
<!-- ✅ 正确：使用 a 标签 -->
<a href="/index.html" class="btn">🏠 首页</a>

<!-- ❌ 不推荐：使用 button + JS -->
<button onclick="location.href='/index.html'">🏠 首页</button>
```

**原因**:
- 语义化更好（链接就是导航）
- SEO 友好
- 支持右键打开新标签
- 支持中键点击
- 支持拖拽链接

### CSS 样式

**链接特殊处理**:
```css
.btn {
    text-decoration: none;  /* 去掉下划线 */
    cursor: pointer;
}
```

## ⚠️ 注意事项

### 1. 相对路径

使用相对路径 `/index.html` 而非绝对路径：
- ✅ `/index.html`
- ❌ `http://localhost:3000/index.html`

**原因**: 部署到生产环境时自动适配域名

### 2. 样式一致性

确保首页按钮与其他按钮样式一致：
- 相同 padding
- 相同颜色
- 相同圆角
- 相同悬停效果

### 3. 可访问性

- 使用语义化的 `<a>` 标签
- 添加清晰的文字（"首页"）
- 使用通用图标（🏠）
