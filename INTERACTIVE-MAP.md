## 🗺️ 新增交互功能

### 1. 脉冲动画效果 💫

**效果**:
- 每个省份标记都有呼吸脉冲效果
- 脉冲半径周期性扩大缩小
- 视觉引导，快速定位省份

**技术实现**:
```javascript
// 双层标记
const marker = L.circleMarker(...);  // 主标记
const pulse = L.circleMarker(...);   // 脉冲层

// 脉冲动画
setInterval(() => {
    pulse.setRadius(baseRadius + 5);
    setTimeout(() => {
        pulse.setRadius(baseRadius);
    }, 1000);
}, 2000);
```

**交互**:

## 🎯 使用方式

### 基础操作

| 操作 | 效果 |
|-----|------|
| 点击标记 | 打开弹窗 + 选中省份 |
| 鼠标悬停 | 标记高亮放大 |
| 滚轮滚动 | 地图缩放 |
| 拖动地图 | 平移视图 |
| 双击地图 | 放大一级 |

### 高级功能

#### 1. 快速定位

## 📊 视觉效果

### 标记大小

标记大小与总分成正比：
```
总分越高 → 标记越大
80 分 → 半径 16px
60 分 → 半径 14px
40 分 → 半径 12px
```

### 颜色编码

| 等级 | 颜色 | 含义 |
|-----|------|------|
| A 级 | 🟢 绿色 | 优先投资 |
| B 级 | 🔵 蓝色 | 推荐投资 |
| C 级 | 🟡 黄色 | 谨慎投资 |
| D 级 | 🔴 红色 | 暂不建议 |

### 动画效果

**脉冲动画**:
- 周期：2 秒
- 幅度：半径 +5px
- 缓动：平滑过渡

**淡入动画**:
- 时长：0.5 秒
- 效果：从下到上淡入
- 延迟：按索引递增

## 🔧 技术实现

### 标记分层

```javascript
// 主标记（实心）
L.circleMarker(coords, {
    fillColor: color,
    fillOpacity: 0.7,
    radius: baseRadius
});

// 脉冲层（空心）
L.circleMarker(coords, {
    color: color,
    fillOpacity: 0,
    opacity: 0.3
});
```

### 弹窗定制

```javascript
const popup = L.popup({
    maxWidth: 300,
    className: 'province-popup'
});

popup.setContent(`
    <div class="custom-popup">
        <!-- 详细内容 -->
    </div>
`);
```

### 事件监听

```javascript
// 悬停高亮
marker.on('mouseover', function() {
    this.setStyle({ weight: 4, radius: larger });
});

marker.on('mouseout', function() {
    this.setStyle({ weight: 2, radius: normal });
});

// 点击联动
marker.on('click', () => {
    selectProvince(province);
    this.openPopup();
});
```
