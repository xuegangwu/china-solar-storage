## 🐛 问题描述

用户反馈：中国电站投资地图页面显示不对

**症状**:
- 省份列表显示"加载失败，请检查后端服务"
- 或显示"未找到匹配的省份"
- 地图无法显示省份标记

---

## 🔍 问题诊断

### 问题 1：字段引用错误

**问题代码** (`map-v1.0.html`):
```javascript
// 第 277 行
<div class="province-score">总分：${province.total.toFixed(1)}</div>

// 第 260 行
const sorted = [...filteredProvinces].sort((a, b) => b.total - a.total);

// 第 321 行
document.getElementById('detail-total-score').textContent = province.total.toFixed(1);
```

**实际情况**:
- API 返回的数据结构：`{ scores: { total: 83.5 }, ... }`
- 前端错误引用：`province.total`
- 正确引用：`province.scores.total`

### 问题 2：筛选列表未初始化

**问题代码** (`map-v1.0.html` 第 240-245 行):
```javascript
if (data.provinces && data.provinces.length > 0) {
    provincesData = data.provinces;
    // ❌ 缺少：filteredProvinces = [...provincesData];
    renderProvinceList();
    renderMap();
    updateStats();
}
```

**影响**:
- `filteredProvinces` 初始值为空数组 `[]`
- 加载数据后未初始化，仍为空
- `renderProvinceList()` 渲染空列表，显示"未找到匹配的省份"

---

## ✅ 修复方案

### 修复 1：字段引用

**文件**: `web/map-v1.0.html`

**修改命令**:
```bash
sed -i 's/province\.total/province.scores.total/g' map-v1.0.html
```

**影响位置**（4 处）:
- 第 260 行：排序逻辑
- 第 277 行：列表分数显示
- 第 321 行：详情面板分数
- 第 353、369 行：地图标记和弹窗

### 修复 2：初始化筛选列表

**文件**: `web/map-v1.0.html`

**修改**:
```diff
  if (data.provinces && data.provinces.length > 0) {
      provincesData = data.provinces;
+     filteredProvinces = [...provincesData]; // 初始化筛选列表
      renderProvinceList();
      renderMap();
      updateStats();
      document.getElementById('loading').style.display = 'none';
  }
```

## ⚠️ 注意事项

### 数据结构一致性

API 返回的省份数据结构：
```json
{
  "id": 1,
  "name": "上海市",
  "scores": {
    "total": 83.5,
    "resource": 25,
    "policy": 20,
    "market": 18,
    "grid": 20.5
  },
  "grade": "B",
  "country": "CN",
  ...
}
```

**重要**:
- `total` 在 `scores` 对象内，不在根级别
- 所有使用 `province.total` 的地方都要改为 `province.scores.total`

### 其他可能受影响的页面

检查所有使用 `province.total` 的文件：
