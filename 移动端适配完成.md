## 🔧 适配内容

### 1. 通用移动端 CSS

**文件**: `web/mobile-responsive.css`

**核心适配**:

#### 平板适配 (≤ 768px)
```css
@media (max-width: 768px) {
    /* 容器调整 */
    .container { padding: 20px 15px; }
    
    /* Header 调整 */
    .header {
        flex-direction: column;
        gap: 15px;
    }
    
    /* Sidebar 调整 */
    .sidebar {
        width: 100%;
        max-height: 40vh;
    }
    
    /* 地图区域 */
    .map-area { height: 60vh; }
    
    /* 统计卡片 */
    .stats-grid { grid-template-columns: repeat(2, 1fr); }
}
```

#### 手机适配 (≤ 480px)
```css
@media (max-width: 480px) {
    /* 字体进一步缩小 */
    .header h1 { font-size: 18px; }
    .header p { font-size: 11px; }
    
    /* 按钮堆叠 */
    .header-actions {
        flex-wrap: wrap;
        justify-content: center;
    }
    
    /* 卡片内容 */
    .card { padding: 15px; }
}
```

#### 横屏模式优化
```css
@media (max-height: 500px) and (orientation: landscape) {
    .sidebar { max-height: 150px; }
    .map-area { height: calc(100vh - 150px); }
}
```

## 📋 适配要点

### 1. Viewport 设置

所有页面已包含正确的 viewport 设置：
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 2. 响应式单位

- 使用 `rem` 和 `em` 而非固定 `px`
- 使用百分比 `%` 和 `vw/vh`
- 使用 `max-width` 而非固定宽度

### 3. 触摸优化

- 按钮最小点击区域 44x44px
- 增加触摸反馈
- 禁用双击缩放

### 4. 性能优化

- 减少移动端不必要的动画
- 优化图片加载
- 懒加载长列表

## ⚠️ 注意事项

### 1. 浏览器兼容性

**推荐浏览器**:
- ✅ Chrome 90+ (Android/iOS)
- ✅ Safari 14+ (iOS)
- ✅ Firefox 88+ (Android)
- ✅ Edge 90+ (Android/iOS)

### 2. iOS 特殊处理

```css
/* 防止 iOS 自动调整字体大小 */
body {
    -webkit-text-size-adjust: 100%;
}

/* 防止 iOS 点击高亮 */
* {
    -webkit-tap-highlight-color: transparent;
}
```

### 3. Android 特殊处理

```css
/* 防止 Android 地址栏影响布局 */
html, body {
    height: 100%;
    overflow: hidden;
}
```
